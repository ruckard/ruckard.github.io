Zignaly Tools

  • Zignaly Signal Wizard. The Zignaly Signal Wizard let's you build the signals you want to send to Zignaly from TradingView (or other systems) so that trades are then forwarded to your exchange account thanks to Zignaly. 
    "pair":"BTCUSDT",
"signalId":"long-BTCUSDT",
"exchange":"zignaly",
"exchangeAccountType":"futures",
"side":"long",
"leverage":"1",
"type":"entry",
"orderType":"market",
"positionSizePercentage":"50.0",
"key":"MYSECRETKEY"
  • CopyTrader - First steps. CopyTrader - First steps teaches you from the start how to interact with Zignaly in order to become a CopyTrader either for others to copy your or for your own trading.
    • Sign-up as a Zignaly platform user.
    • Create a Copy-Trade service in Zignaly.
    • Create an exchange account in Zignaly with the same Exchange (e.g. Binance) and Exchange type (e.g. futures) that you will be offering as a provider.
